VITA 스마트홈 - 구현 기능 정리 (서비스 제안서용)
작성일: 2026-07-16

==================================================
서비스 개요
==================================================
VITA는 에너지 절약과 화재 안전을 한 앱에서 통합 관리하는 스마트홈 서비스입니다.
방과 기기를 등록해 실시간으로 제어하고, 전력 사용량·전기요금을 예측하며, 캘린더 일정과
재실 여부에 맞춰 조명·온도를 자동으로 조절합니다. 동시에 온도·습도·연기감지 센서로 화재
징후를 실시간 감시해 자동 차단·알림·119 신고 안내까지 제공하는 것이 핵심 차별점입니다.


==================================================
1. 홈 대시보드
==================================================
- 실시간 시계와 요일을 큰 화면으로 표시.
- 내부 습도 · 실내 온도 · 날씨를 한 화면에서 확인.
- "오늘의 절전 목표" 달성률을 진행 막대(러너 아이콘)로 시각화.
- 스마트홈 제어 · 캘린더 · 에너지 사용량 · 에너지 나무로 바로 이동하는 4대 바로가기.
- 알림(종 아이콘, 읽지 않은 알림 배지) · 설정 · 전체 메뉴를 한 화면에서 접근.


==================================================
2. 스마트홈 기기 제어
==================================================
- 방(공간) 단위로 기기를 자유롭게 등록·삭제하고, 방 이름도 직접 관리.
- 기기별 "자동/수동" 모드: 평소엔 센서 값을 그대로 반영하고, 센서 이상 시에는 사용자가
  직접 전원을 켜고 끌 수 있음.
- 방마다 "목표 온도" 설정 및 −/+ 버튼으로 즉시 조절.
- 현재 켜져 있는 기기 수를 한눈에 확인하는 전체 현황 카드.


==================================================
3. 캘린더 & 일정 관리
==================================================
- 매일/특정 요일 반복 일정(루틴)과 날짜 지정 일정(스페셜)을 구분해 등록.
- 스페셜 일정에는 "일반 / 외출 / 외박" 유형을 태그로 지정 가능.
- 등록된 일정은 자동화 규칙의 트리거로 그대로 활용됨(아래 4번 참고).


==================================================
4. 일정·재실 기반 자동화 규칙
==================================================
- "자동화 규칙" 화면에서 사용자가 직접 규칙을 만들고 관리.
- 트리거 4종:
  · 외출 예정 (캘린더에 등록한 외출 일정)
  · 외박 일정 (캘린더에 등록한 외박 일정)
  · 요일별 루틴 (캘린더의 반복 일정, 특정 요일만 선택 가능)
  · 재실/외출 감지 (사람이 집에 있는지 여부에 따라 실시간 반응)
- 실행 조건: 일정 시각 기준 "몇 분 전"에 실행할지 직접 지정 가능(외출/외박/루틴 트리거).
- 실행 동작: 지정한 방의 기기를 켜기/끄기, 또는 목표 온도를 원하는 값으로 설정.
- 재실/외출 트리거는 "재실일 때 희망 온도"와 "외출 중일 때 희망 온도"를 각각 설정해두면,
  재실 상태가 바뀌는 즉시 해당 방의 목표 온도가 자동으로 전환됨.
- 재실 여부는 카메라 인식으로 판단할 예정이며, 현재는 화면의 스위치로 직접 시뮬레이션해
  전체 동작(감지→온도 변경→알림)을 미리 확인할 수 있음. 카메라 연동 시 감지 결과만
  이어붙이면 되도록 구조화됨.
- 화면을 보고 있지 않아도 백그라운드에서 계속 감시하며 조건 충족 시 즉시 실행 + 알림 발송.


==================================================
5. 화재 감지 및 자동 안전 대응
==================================================
- AI 이상 패턴 감지: 기기 종류별 "정상 사용 시간"을 기준으로, 그보다 오래 켜져 있으면
  이상으로 판단해 자동으로 전원 차단(히터·전기장판·가스레인지 등 고위험 기기는 우선 감지).
- 온도 · 습도 · 연기감지 센서: 방마다 실시간 센서 값을 표시하고, 연기 감지 또는 급격한
  고온 발생 시 원인 기기를 특정할 필요 없이 그 방의 전원을 즉시 전체 차단.
- 두 감지 방식 중 더 위험한 쪽을 기준으로 방별 위험도(안전/주의/위험)를 종합 판정.
- 위험 감지 시 긴급 경보 배너 표시 + 119 신고 연결 버튼(전화 앱 자동 실행) + 자동 대응
  이력 기록 + 앱 알림 발송까지 한 번에 처리.
- 실제 센서가 연결되기 전까지는 더미 데이터로 자동 생성되며, 방마다 "화재 상황 시뮬레이션"
  버튼으로 위험 상황을 직접 재현해 전체 대응 흐름을 테스트해볼 수 있음. 추후 실제 온도/습도/
  연기감지 센서를 연결하면 지금의 감지·자동차단·알림 로직은 그대로 활용 가능.


==================================================
6. 에너지 사용량 분석
==================================================
- 실제 등록된 기기의 켜짐/꺼짐 이력을 실시간으로 누적해 하루 단위 사용량(kWh) 기록.
- 연간/월간/일간 단위로 전환하며 사용량 추이를 라인 차트로 확인.
- 전년/전월/전일 대비 증감률을 자동 계산해 한눈에 비교.
- 방과 무관하게 기기 종류별로 묶어, 실시간으로 가장 전력을 많이 쓰는 기기 순위를 표시.


==================================================
7. 에너지 나무 (절약 습관 게이미피케이션)
==================================================
- 절전 목표 대비 실제 사용량을 기준으로 나무가 새싹 → 어린 나무 → 다 자란 나무로 성장.
- 날짜별 성장 기록을 달력형 "성장 트래커"로 확인.
- 한 달 전체를 모아보는 "숲 보기"(월말엔 자동으로 표시).
- 이번 달 절약량(kWh)과 CO2 절감량을 함께 통계로 제공해 환경적 의미도 전달.


==================================================
8. 전기요금 예측 (영수증 미리보기)
==================================================
- 현재 켜져 있는 기기 목록과 실시간 소비전력(W)을 표시.
- 오늘까지의 사용 패턴을 기준으로 이번 달 예상 사용량과 진행률을 계산.
- 누진 요금 구간을 반영한 예상 전기요금 영수증 미리보기 제공.


==================================================
9. 안전 가이드북
==================================================
- 화재 예방 생활수칙, 우리 집 위험도 자가진단 체크리스트, 상황별 초기 대응 요령,
  자취생 꿀팁, 아이와 함께 배우는 화재 안전, 대피 계획 세우기, 비상용품 체크리스트,
  화재 진압 후 해야 할 일, 화재 관련 안전 상식, 긴급 연락처까지 총 11개 섹션으로 구성된
  화재 안전 교육 콘텐츠.
- 가족 비상 연락처와 경비실(관리사무소) 번호를 직접 등록해두면 탭 한 번으로 바로 전화 연결.
- 위험도 체크리스트는 체크한 항목 수에 따라 "안전/주의/개선 필요"로 자동 진단.


==================================================
10. 알림 · 설정
==================================================
- 자동화 실행, 화재 위험 감지/자동 차단 등 시스템이 스스로 발생시키는 이벤트를 알림으로
  실시간 수신. 읽음 처리 및 삭제 관리 가능.
- 설정 화면에서 집 주소 등록, 안전 가이드북 전용 글자 크기 조절(작게/보통/크게), 앱 정보 확인.


==================================================
현재 프로토타입 단계 참고사항
==================================================
아래 항목들은 실제 하드웨어/외부 API 없이 더미 데이터·규칙 기반 시뮬레이션으로 동작하도록
구현되어 있으며, 실제 연동 시 해당 부분만 교체하면 전체 로직을 그대로 재사용할 수 있도록
설계되어 있습니다.
- 온도 · 습도 · 연기감지: 지금은 더미 값으로 자동 생성 → 실제 센서/IoT 장비 연동 예정.
- 재실/외출 판단: 지금은 수동 스위치로 시뮬레이션 → 실제 카메라 인식 결과 연동 예정.
- 전기요금 계산: 실제 스펙/한전 API 대신 근사치 사용 → 실제 요금제/전력 데이터 연동 시 정교화 예정.
- 화재 위험도(AI 이상 패턴 감지): 학습된 모델이 아닌 기기 사용 시간 기준 규칙 기반 로직.
